# Kafka Topic Producer/Consumer Map

이 문서는 현재 코드 기준의 Kafka topic 발행/수신 연결표입니다. Producer 는 각 서비스의
`adapter/out/messaging/*KafkaEventProducer`, Consumer 는
`adapter/in/messaging/consumer/*EventConsumer`를 기준으로 정리했습니다.
